# 小程序端功能
## 一、登录与注册功能
	微信首次进入授权，获取用户手机号和真实姓名并保存至用户表 (接口)
	微信授权登录 (接口)
	整体权限验证 (在其它页面如果无用户信息自动跳转至登录页)

	页面0与接口调用

## 二、运动功能
	功能展示页面(页面1)
	走路(页面3)
	慢跑(页面4)
	运动结束页面 (页面5&页面6)
	运动数据上传，对齐数据结构（接口）

## 三、测评表
	数据库写死题库前端直接渲染
	保存用户做题记录（接口）
